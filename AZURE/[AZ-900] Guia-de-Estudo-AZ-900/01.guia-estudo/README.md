# Guia de estudo: Microsoft Azure Fundamentals (AZ-900)

##  1. Conceitos de Nuvem (25-30%)
* **Tópicos chave:** benefícios da nuvem (Alta Disponibilidade, Escalabilidade, Agilidade, Elasticidade), CapEx vs OpEx, e Modelos de Serviço (IaaS, PaaS, SaaS).
* **🔗 Microsoft Learn:** [https://learn.microsoft.com/pt-br/training/paths/microsoft-azure-fundamentals-describe-cloud-concepts/]


## 2. Arquitetura e serviços do Azure (35-40%)
* **Tópicos chave:** estrutura física (Regiões, Zonas de Disponibilidade), Recursos (VMs, App Services, Container Instances), e Redes (VNET, ExpressRoute, VPN Gateway).
* **🔗 Microsoft Learn:** [https://learn.microsoft.com/pt-br/training/paths/azure-fundamentals-describe-azure-architecture-services/]


## 3. Gerenciamento e governança (30-35%)
* **Tópicos chave:** custo (Calculadora de TCO, Cost Management), Governança (Azure Policy, Blueprints), ferramentas (Portal, CLI, PowerShell, ARM Templates).
* **🔗 Microsoft Learn:** [https://learn.microsoft.com/pt-br/training/paths/describe-azure-management-governance/]
