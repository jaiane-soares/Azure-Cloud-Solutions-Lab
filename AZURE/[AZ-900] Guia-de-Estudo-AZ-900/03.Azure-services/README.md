
### 📋 Quadro de alguns serviços do Azure (são muitos...) `meu amigo comentou que existe um Azure Service Bus essa semana (como assim eu não sabia desse??)`

| Categoria | Serviço Azure | Descrição (Foco AZ-900) |
| :--- | :--- | :--- |
| **Computação** | **Azure Virtual Machines (VM)** | **IaaS**: Máquinas virtuais sob demanda. Oferece controle total sobre o SO e software. |
| **Computação** | **Azure App Service** | **PaaS**: Hospedagem de aplicações web e APIs. O Azure gere a infraestrutura e o runtime. |
| **Computação** | **Azure Functions** | **Serverless**: Execução de código baseada em eventos (triggers) sem gerir servidores. |
| **Computação** | **Azure Container Instances (ACI)** | Método mais rápido para rodar um contentor no Azure sem gerir VMs. |
| **Rede** | **Azure Virtual Network (VNet)** | Permite a comunicação segura entre recursos Azure e isolamento de rede local. |
| **Rede** | **VPN Gateway** | Envia tráfego encriptado entre uma rede virtual Azure e um local físico (On-premises). |
| **Rede** | **Azure ExpressRoute** | Ligação privada e dedicada de alta velocidade entre a tua infraestrutura e o Azure. |
| **Armazenamento** | **Azure Blob Storage** | Armazenamento de objetos para dados não estruturados (imagens, vídeos, ficheiros). |
| **Armazenamento** | **Azure Files** | Partilhas de ficheiros na nuvem totalmente geridas (acedidas via SMB ou NFS). |
| **Bases de dados** | **Azure SQL Database** | Base de dados relacional (PaaS) totalmente gerida e inteligente. |
| **Bases de dados** | **Azure Cosmos DB** | Base de dados NoSQL distribuída globalmente com baixa latência e alta escala. |
